```yaml

  - brand: "小米"                     # 品牌名称：商品所属的品牌
    product_name: "小米手环8 Pro"      # 商品名称：完整的产品名称，用于展示或检索price: 299.00                    # 售价：单位为元（人民币），使用浮点数表示product_url: "https://example.com/xiaomi-band-8-pro"  # 商品链接：commissions:                     # 佣金比例信息列表

      rate: 0.10                   # 投流佣金比例
      roi: 0.15                    # 投流ROI，只有达到 ROI 才会复投，如果商家说不卡 roi，这个字段为 0

      rate: 0.05                   # 不投流佣金比例
    wx_promot: ""                  # 视频号投流方式，可选值包括不支持视频号、视频号纯佣、视频号同步投流
```

注意:
1. 如果对话没有明确说明，投流和不投流的佣金比例，或者一个产品只有一个佣金率，默认只支持投流。
2. 商品信息只可能来自商家，不可能来自我
3. 是否寄样, 单号不需要精确匹配产品
4. 是否寄样，只有我明确提出寄样，才认为是寄样，商家表述要寄样不算。在 import_desc 字段记录，你是如何判断的。
5. wx_promot 用于描述商品是否支持视频号投流，通常用户会使用 "平台:" 的标识说明视频号投流的方法。我也有可能会直接问商家是否支持视频号投流。需要从这两个信息中提取视频号投流的方式。
